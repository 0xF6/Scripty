## Build or run

* Clone the repository
* cd to src
* `$ dotnet run .` to run
* `$ dotnet publish -c release -r <TARGET> -p:PublishSingleFile=true -p:PublishTrimmed=true -p:PublishReadyToRun=true`
  to build
